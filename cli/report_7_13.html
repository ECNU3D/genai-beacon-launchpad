LLM 的阴暗面：基于 Agent 的攻击实现完整计算机接管

作者：Matteo Lupinacci, Francesco Aurelio Pironti, Francesco Blefari, Francesco Romeo, Luigi Arena, Angelo Furfaro 这项开创性的安全研究意义重大，因为它首次全面评估了 LLM agents 作为攻击向量，能够实现完整的计算机接管。虽然大多数 LLM 安全问题都集中在 prompt injection 上，但这项工作表明，agentic 系统引入了全新的漏洞类别，可能允许恶意行为者突破信任边界并获得未经授权的系统访问权限。作者引入了一种新颖的攻击框架，其中 LLM agents 被武器化，用于在目标系统中执行侦察、利用和持久化。通过对多个 agent 框架进行系统测试，他们展示了利用 LLM agents 的自主性绕过传统安全控制的成功攻击。这项研究对 agentic AI 系统日益广泛部署所带来的安全影响发出了严峻警告。

[来源]

DOTResize：通过基于离散最优传输的神经元合并来减小 LLM 宽度

作者：Neha Verma, Kenton Murray, Kevin Duh 机构：Johns Hopkins University 本文通过引入一种针对宽度而非深度缩减的模型压缩新方法，解决了 LLM 部署中的一个关键挑战。DOTResize 将神经元压缩框定为一个离散最优传输问题，智能地合并相似神经元以减小模型宽度，同时保持性能。这种方法特别重要，因为它提供了一个数学框架，用于识别和消除大型语言模型中的冗余，而无需重新训练。作者证明，他们的方法可以将 LLM 层的宽度减少高达 50%，在各种基准测试中性能下降极小。通过专注于宽度缩减，DOTResize 补充了现有的剪枝和量化技术，有望在资源受限的设备上实现更高效的大型语言模型部署。
